# cp-vinheria
Dispositivo utilizado para monitorimento de ambiente, buscando um local ideal para manter um acervo de vinhos

O aparelho apresentado nesta página tem a função de monitorar o espaço de uma sala para que a ambientação esteja em um estado ideal para a conservação em um acervo de uma vinheria, levando em conta umidade, temperatura e luminosidade, já que são condições que afetam a qualidade do vinho diretamente. O projeto se utiliza de peças como arduino, DHT11, rtc ds1307, resistores, tela LCD, buzzer e LEDs. Para o uso do aparelho, é necessário o colocar na mesma sala em que se mantém os vinhos, caso haja problemas com a climatização da sala, um alerta sonoro será ativado, acompanhado de uma luz vermelha. As luzes LED indicam a qualidade do ambiente, sendo a verde um estado OK, amarelo um estado de alerta e vermelho um estado crítico. Informações também são exibidas através da tela LCD para o usuário.

Junto ao código gravado no arduino, estão disponibilizados um vídeo explicativo, imagem do circuito e o projeto simulado no Wokiwi
